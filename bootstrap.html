frontend

the development of the graphical user interface of a website through the use of HTML, CSS, and JavaScript so users can view and interact with that website.!

HTML

HTML, or HyperText Markup Language, is the code that defines the structure and content of a web page.

It's the most fundamental building block of the web. .

CSS

Selectors are CSS identifiers that specify which HTML components to decorate throughout the web page

There are three types of CSS: external, internal, and inline.

JAVASCCRIPT

JavaScript is a programming language that adds interactivity to web pages..

JavaScript is a key part of web development, along with HTML and CSS.
